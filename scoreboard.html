Web Bluetooth not available

On Linux / Chrome: Web Bluetooth is disabled by default.
Go to chrome://flags/#enable-experimental-web-platform-features, set it to Enabled, then relaunch Chrome.

On Android: use Chrome or Edge — works out of the box.
iOS Safari: not supported (Apple restriction).
Firefox: not supported.
